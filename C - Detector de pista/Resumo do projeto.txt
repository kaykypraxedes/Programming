O programa processa várias sequências de números (chaves), classificando-os em tipos (1, 2, 3) e agrupando valores consecutivos iguais para otimização. 

Uma vez os valores agrupados, o programa busca pela sequência (1, 3, 2, 3, 1) para determinar pista, verifica a varição do ponto médio de cada linha para identificar curvas, além de  identificar padrões que indicam impedimentos na pista.

O sistema utiliza listas encadeadas com alocação dinâmica para  realizar a a análise independente de cada linha sequência e ao final retorna a interpretação dos dados para quem estiver.
